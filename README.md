# AI Diffusion State

This repository hosts the empirical pipeline for **The Diffusion State: China’s AI Industrialization Model and the Next Productivity Shock**.

The project measures whether China’s AI advantage may lie less in frontier-model leadership and more in state-guided diffusion into industry.

Core empirical chain:

```text
AI pilot zones -> smart-factory adoption -> industrial AI diffusion -> export and productivity upgrading
```

Start here:

1. `docs/PROJECT_EXECUTION_PLAN.md`
2. `docs/STANDALONE_ENGINEER_INSTRUCTIONS.md`

The working principle is strict reproducibility: raw files are preserved, processed rows carry source provenance, inferred fields carry confidence flags, and paper tables are generated by scripts.
